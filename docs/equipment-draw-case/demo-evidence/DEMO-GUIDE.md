## 1. Overview

Equipment Draw Demo is an F4-triggered three-choice equipment selection system built on the shared `selection_pool + selection_outcome` family seam.

Prompt:

> Create an equipment draw system triggered by F4. When the player presses F4, draw 3 candidates from a weighted equipment pool, show a card selection UI, let the player confirm one equipment, deliver the native item immediately, and make selected equipment not appear again in future draws during the same session.

The demo uses explicit source-backed authoring parameters. The generic Wizard is checked before fixture merge and must not inject case-owned selection parameters.
